# 🎓 Udemy Clone

A responsive front-end clone of the Udemy homepage built using **HTML5** and **CSS3**. This project replicates the core layout and user interface of Udemy, focusing on clean design, semantic HTML, and responsive web development.

## 📌 Features

- 🔍 Responsive navigation bar with search box
- 📚 Course categories menu
- 🎯 Recommended Courses section
- 🏷️ Recommended Topics section
- ⭐ Popular Courses with ratings and reviews
- 💰 Course pricing with discounts
- 📱 Mobile-friendly responsive design
- 📄 Footer with useful links

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Font Awesome
- Google Fonts

## 🎯 Project Objective

The objective of this project was to recreate the Udemy homepage while improving my front-end development skills. It helped me practice semantic HTML, responsive layouts, Flexbox, CSS Grid, and modern UI design techniques.
